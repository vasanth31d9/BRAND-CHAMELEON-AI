                            🆘 NEED HELP?

═══════════════════════════════════════════════════════════════════════════

Check these URLs:
• Frontend: http://localhost:3000
• Backend:  http://localhost:8000
• API Docs: http://localhost:8000/docs

Check logs:
• Frontend: Browser console (F12)
• Backend:  Terminal where backend is running

Common issues:
• "Module not found" → Run npm install or pip install
• "Port in use" → Kill process on port 3000/8000
• "Invalid API key" → Check .env files for typos
• "Can't login" → Verify Supabase keys are correct
